Game Producing

Introduction to My Research Papers

This page introduces research related to game producing, game design, player experience analysis, and game analysis. The research includes methods for analyzing player interest using brain waves and biological information, supporting game design through gameplay analysis, and improving user interfaces and game experiences.

Game Design

Game Design Based on Brain-Wave Analysis

This research investigates how brain-wave analysis can be used to understand and manipulate player interest levels. It aims to connect physiological data with game design methods and player experience.
